Для первой игры:

1) usefulness_course (матрицу весов w )- индив полезность предметов (для карьеры) для каждого студента. Размерность n*m
2) quality_teaching (вектор весов a) - полезность курса, которая зависит от преподавателя. Размерность m
3) ease_passing_exam (вектор весов d) - легкость сдачи курсов, которая зависит от преподавателя. Размерность m
4) student_laziness (вектор весов b) - (от 0 до 1)  индивидуальный компромисс студента i между академической ценностью и легкостью сдачи (bi=1: важна только академическая ценность; bi=0: важна только легкость)
5) importance_certainty (вектор  весов alpha) - коэффициент важности определенности информации.
6) exchange_rate_uncertainty (вектор весов omega) - уровень неопределенности курса j (omega_min для нового, omega_high для нового курса)
7) capacity (вектор весов c_max) - максимальная вместимость курсов
8) min_quantity (вектор весов c_min) - минимальный порог для i курса
9) student_certainty_weight (вектор весов lambda) - коэффициент важности определенности информации
